## 📌 Overview

JanSahayak AI is a multilingual, AI-powered civic assistant designed to simplify access to Indian government welfare schemes. Powered by Google Gemini 2.5 Flash, it helps citizens discover relevant schemes, understand eligibility requirements, and prepare necessary documents through natural language conversations.

The platform reduces bureaucratic complexity and makes public welfare information more accessible, especially for citizens facing language and digital literacy barriers.

---

## 🛑 Problem Statement

Millions of eligible citizens fail to access government benefits because:

* They do not know which schemes apply to them.
* Government portals are often difficult to navigate.
* Information is primarily available in English or formal Hindi.
* Eligibility criteria and documentation requirements can be confusing.

---

## 💡 Solution

JanSahayak AI acts as a digital government assistance companion.

Users simply provide basic details such as:

* Age
* Income
* Occupation
* State

The AI then:

* Recommends relevant government schemes
* Explains eligibility requirements
* Lists required documents
* Provides official resources and guidance
* Responds in the user's preferred language

---

## ✨ Key Features

### 🎯 Scheme Discovery Engine

Matches users with relevant Central and State Government schemes based on their profile and circumstances.

### 📄 Document Readiness Analyzer

Checks document preparedness, identifies missing requirements, and guides users through obtaining essential documents.

### 💬 AI Citizen Assistant

Answers civic and government-related questions with concise, actionable responses.

## 🚀 Getting Started

### 1. Clone the Repository

```bash
https://github.com/ArixenX/JanSahayak-AI.git
```

### 2. Configure Gemini API Key

Get a free API key from [Google AI Studio](https://aistudio.google.com/).

Open `app.js` and replace:

```javascript
const GEMINI_API_KEY = "YOUR_API_KEY";
```

with your actual API key.

### 3. Run the Application

Open `index.html` in any modern web browser.
